github簡介

0922的日子

初始設定

$git config --global user.name "使用者名稱"

$git config --global user.email "使用者信箱"

下載檔案

$git clone "網址"

要先更改檔案位置 $ cd /d/..../

上傳檔案 (要把當前位置移動到要上傳的檔案位置) 工作目錄->Stage $git add "檔名"

$git status 查看檔案修改狀態

Stage -> Repo $git commit -m "修改原因或其他備註"

更新到 Github $git push 上傳到github

*在察看狀態時，紅色字顯示的是修改過的檔案 (註):$ pwd 查看當前位置 $cd /磁區/資料夾/子資料夾... 改變當前位置
